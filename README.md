# BodyBalanceBites
This repo is for the meal planner project under IBM Steps to run:

Download the web project.
Install a text editor or IDE and a web browser.
If needed, install Node.js.
Navigate to the project directory in the terminal.
Install local development server package by command "npm i".
Start the local development server with the command "npm run start".
Open the project in the web browser given by the server's specified URL.
Interact with the web project in the browser.
Stop the local development server using Ctrl+C in the terminal when done.
